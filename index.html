Запустите форум за 5 простых шагов

Никаких технических знаний не требуется — только 1 минута вашего времени

1

GitHub аккаунт

Зарегистрируйтесь бесплатно на github.com

2

Откройте шаблон

github.com/feekool/feekool.github.io

3

Создайте репозиторий

Нажмите "Use this template" → "Create a new repository". Назовите, например, "my-baking-forum"

4

Включите GitHub Pages

Settings → Pages → ветка main → Save. Через 60 секунд ваш форум онлайн!

5

Наполняйте контентом

Добавляйте категории: "Рецепты", "Мастер-классы", "Вопросы", "Галерея". Всё под вашим контролем 🎉
